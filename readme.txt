About RayTrace
	Its a flash logger made in haXe. This can be used as a separate logger along with the utility class present in src/libForApp/ folder named "raytrace.as".
 Usage :
 	1. Put the "raytrace.as" file in your flash application root folder.
 	2. From anywhere in your application write raytrace("message") to see the output in external logger.

 Lets haXe it.
